# OmniSphere Global Retail Profitability Deep Dive

OmniSphere Global 2025: Unlocking Retail Profitability — Drivers, Leaks, High-Value Customers & Margin Strategies

This repository features a comprehensive **Profitability Deep Dive** for Omnisphere, a global retail entity. The analysis identifies key drivers of profit and loss across international markets to support data-driven fiscal strategies.

## 📖 Project Overview
The core objective was to move beyond surface-level sales figures to understand **true profitability** by analyzing the intersection of shipping costs, regional overheads, and product margins.

[![Kaggle](https://img.shields.io/badge/Kaggle-View_Notebook-00BBFF?style=for-the-badge&logo=kaggle)](https://www.kaggle.com/code/joylorna/omnisphere-global-retail-profitability-deep-dive)

## 🛠️ Technical Stack
This project utilizes a dedicated Business Intelligence stack to transform raw relational data into executive-level insights:

* **Data Extraction:** **SQL** – Used for relational data retrieval, complex multi-table joins, and defining core profitability KPIs at the database level.
* **Data Analysis:** **Microsoft Excel** – Employed for data validation, handling missing values, and prototyping comparative performance models.
* **Business Intelligence:** **Tableau** – Leveraged to build interactive executive dashboards for spatial and categorical performance tracking.

---

## ⚙️ The Analytical Process
1. **Understand the Problem:** Defined **KPIs** focused on global gross profit, regional revenue velocity, and shipping cost impact on margins.
2. **Collect the Data:** Extracted structured records from a relational database using optimized **SQL queries**.
3. **Clean and Prepare:** Validated data quality in **Excel**, standardized regional formats, and calculated derived financial metrics.
4. **Analyze and Visualize:** Developed executive-level dashboards in **Tableau** to identify underperforming regions and high-margin product categories.
5. **Interpret and Communicate:** Translated findings into **strategic recommendations** for supply chain optimization and regional pricing.
